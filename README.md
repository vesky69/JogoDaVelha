# Jogo da Velha

Este é um projeto simples de Jogo da Velha (Tic-Tac-Toe) implementado em C# usando Windows Forms.

## Descrição

O jogo permite que dois jogadores alternem turnos para marcar X ou O em uma grade 3x3. O objetivo é conseguir três marcas em linha (horizontal, vertical ou diagonal) antes do oponente.

## Funcionalidades

- Interface gráfica simples com botões para cada célula da grade.
- Alternância automática entre jogadores X e O.
- Detecção de vitória ou empate.
- Reinício do jogo após cada partida.

## Requisitos

- .NET Framework ou .NET Core instalado.
- Visual Studio ou outro IDE compatível com C# e Windows Forms.

## Como executar

1. Abra o arquivo `WinFormsAppJulia.slnx` no Visual Studio.
2. Compile e execute o projeto.
3. A interface do jogo será exibida, permitindo jogar imediatamente.

## Estrutura do Projeto

- `WinFormsAppJulia.slnx`: Arquivo de solução.
- `WinFormsAppJulia/Form2.cs`: Código principal do formulário.
- `WinFormsAppJulia/Form2.Designer.cs`: Designer do formulário.
